# AI Study Assistant

An AI-powered study assistant built using Streamlit and the Gemini API.

## Features

- Upload PDF study materials
- Generate summaries
- Create flashcards
- Generate MCQs
- Identify important questions

## Technologies Used

- Python
- Streamlit
- Gemini API
